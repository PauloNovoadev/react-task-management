# React Task Manager

A task management web application built with React. Users can create, read, update, and delete tasks in a simple and dynamic environment.

# Features
- Create tasks
- Edit task titles
- Update task status (Pendente, Executando, Finalizado)
- Delete tasks
- Multi-column layout
- Fixed top navigation bar

# Built With
- React
- Vite
- JavaScript
- CSS
- HTML

# Live Demo
https://react-task-management-three.vercel.app/

# Getting Started

# Install dependencies
```bash
npm install
```
# Run the Project
npm run dev

# Translation(extra)
"Pendente" -> Pending
"Executando" -> In Progress
"Finalizado" -> Completed
